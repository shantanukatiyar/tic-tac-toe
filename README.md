# 🎮 Tic Tac Toe Game

A modern and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo

👉 https://your-username.github.io/tic-tac-toe/

---

## ✨ Features

* 🧠 Two-player gameplay (X vs O)
* 📊 Scoreboard (tracks wins & draws)
* 🎯 Winning cell highlight
* 🔄 Restart game option
* ♻️ Reset score functionality
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript 

---

## 📂 Project Structure

tic-tac-toe/
│── index.html
│── style.css
│── script.js

---

## ⚙️ How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
